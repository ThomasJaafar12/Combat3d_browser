## Character Coverage

V0 now includes a minimal exception subset of rigged character bodies and animations.

- `Sword and Shield Pack` supplies the leader and tank/disruptor body plus core melee clips.
- `Pro Longbow Pack` supplies the ranged companion and ranged enemy body plus core bow clips.
- `Pro Melee Axe Pack` supplies the melee chaser and melee companion body plus core axe clips.
- These packs remain heavy compared with the rest of the V0 subset, so only a narrow body-and-animation slice was copied.
- The current melee axe subset still lacks a curated death clip because the selected local axe pack files do not expose a clean matching death animation in the same compact subset.
