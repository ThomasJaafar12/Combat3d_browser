## Source Asset Policy

`assets/source/` is intentionally almost empty for V0.

- The selected subset prioritizes runtime-ready files in `assets/game/`.
- No editable master files were copied unless they were the only practical runtime option.
- The current weapon set comes from runtime `gltf` exports in the KayKit pack and is curated directly into `assets/game/weapons/`.
- Add files to `assets/source/` only after a concrete conversion or reauthoring task justifies the extra repo weight.
